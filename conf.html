Accueil Historique Confidentialité Services

Confidentialité

  1. Facebook :
    Paramètre Description
    Publications Choisir qui peut voir vos posts (Public, Amis, Personnalisé).
    Publicités ciblées Gérer les publicités en fonction de vos données.
    Applications Contrôler les applications tierces accédant à vos informations.
    Reconnaissance faciale Option pour désactiver l’identification automatique sur les photos.
  2. Instagram :
    Paramètre Description
    Compte privé Permettre uniquement aux abonnés approuvés de voir vos publications.
    Interactions Limiter qui peut commenter ou vous envoyer des messages.
    Publicités ciblées Gérer les publicités que vous voyez en fonction de vos données.
  3. TikTok :
    Paramètre Description
    Compte privé Limiter l'accès à vos vidéos à vos abonnés.
    Commentaires et messages Restreindre qui peut commenter ou vous envoyer des messages privés.
    Publicités ciblées Gérer la collecte de données pour les publicités.
